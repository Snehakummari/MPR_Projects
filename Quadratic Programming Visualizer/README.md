Quadratic Programming Solver

The Quadratic Programming Solver is a graphical application designed to formulate, solve, and visualize quadratic programming (QP) optimization problems using Karush–Kuhn–Tucker (KKT) conditions.
This tool is particularly useful for students and learners who want to understand quadratic optimization concepts through an interactive and visual interface.

Features

Supports both minimization and maximization quadratic programming problems

Provides a GUI-based interface for entering matrices and vectors

Handles optimization problems with up to 4 variables and 4 constraints

Visualizes constraints and optimal solutions in 2D for problems with two variables

Includes a built-in help section with instructions for using the application

Designed with a clean and user-friendly interface using Python and Tkinter
